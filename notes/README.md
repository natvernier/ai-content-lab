# Notes

This folder collects learning notes, reflections and open questions.

It is a place for documenting what changed, what worked, what failed and what should be tested next.

## Possible contents

- learning logs
- research notes
- open questions
- experiment reflections
- terminology notes
- decision notes

## Working principle

Documentation turns experiments into reusable knowledge.

Notes should make progress visible, even when the work is still early.
