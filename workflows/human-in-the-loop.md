# Human-in-the-Loop Workflow for AI-Supported Content Creation

## Purpose

This workflow defines how AI-generated content can be used safely in regulated communication environments.

## Basic flow

1. Define communication intent
2. Select audience and context
3. Generate draft with AI
4. Review for factual accuracy
5. Review for brand and tone
6. Review for legal, ethical or reputational risk
7. Approve, adapt or reject
8. Document learning signals

## Review questions

- Is the content factually correct?
- Is the source basis clear?
- Does the tone fit the audience?
- Could the output create reputational risk?
- Does a human need to make the final decision?
- What signal should be tracked after publication?

## Measurement logic

If repeated corrections appear in the same category, the prompt or source material must be improved.

If approval takes longer than manual drafting, the workflow needs redesign.

If published content performs better with AI support, the pattern should be documented as reusable.
