# Datasets

This folder documents datasets, source structures and data preparation notes used for AI-supported content experiments.

No private, confidential or sensitive data should be stored here.

## Possible contents

- example datasets
- synthetic test data
- source structure notes
- metadata examples
- data cleaning notes
- RAG preparation notes

## Working principle

Useful AI systems depend on useful source material.

This folder is for understanding how information needs to be structured before it can become reliable input for AI workflows.
