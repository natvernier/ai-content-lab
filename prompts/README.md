# Prompts

This folder collects reusable prompt structures for AI-supported content work.

The goal is not to store random prompts, but to document prompt patterns that can be reused, tested and improved over time.

## Possible contents

- editorial prompts
- persona prompts
- RAG prompts
- content review prompts
- tone and format prompts
- prompt improvement notes

## Working principle

A good prompt is not only a text instruction.

It defines the task, context, audience, constraints, source basis, quality criteria and review logic.
